提醒

关于图片无法显示

　　由于dns污染的问题，导致部分小伙伴访问博客文章过程中图片不能加载。这里给出建议：
使用vs code打开C:\Windows\System32\drivers\etc下的hosts文件，并加入以下内容：

# GitHub Start 

192.30.253.112    github.com 
192.30.253.119    gist.github.com
151.101.184.133    assets-cdn.github.com
151.101.184.133    raw.githubusercontent.com
151.101.184.133    gist.githubusercontent.com
151.101.184.133    cloud.githubusercontent.com
151.101.184.133    camo.githubusercontent.com
151.101.184.133    avatars0.githubusercontent.com
151.101.184.133    avatars1.githubusercontent.com
151.101.184.133    avatars2.githubusercontent.com
151.101.184.133    avatars3.githubusercontent.com
151.101.184.133    avatars4.githubusercontent.com
151.101.184.133    avatars5.githubusercontent.com
151.101.184.133    avatars6.githubusercontent.com
151.101.184.133    avatars7.githubusercontent.com
151.101.184.133    avatars8.githubusercontent.com

# GitHub End

　　然后在cmd终端中使用ipconfig/flushdns刷新dns缓存。

关于PAT题目图片小

　　由于时间有限，题目直接放了截图，1600*900分辨率，由于页面样式的限制，所以显示效果不好，可以放大页面到合适的清晰度即可。（刷题的应该有题录的题目了，所以我这个图片其实就是暂时占个位置，将来有时间了慢慢打上去题目，或者就保持这样了）
